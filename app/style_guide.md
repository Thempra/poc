# Style Guide for the Task Management App

## Introduction
This document provides a detailed style guide for the user interface (UI) and user experience (UX) of our task management application. It aims to ensure consistency across all screens, making the app intuitive and user-friendly.

---

## Color Scheme
### Primary Colors
- **Main Blue**: #007BFF - Used as the primary color for brand elements.
- **Secondary Blue**: #6200EA - A darker shade used for call-to-action buttons and accents.
- **Neutral Gray**: #F8F9FA - Light gray used for backgrounds and text to create contrast.

### Accent Colors
- **Green**: #28A745 - Used to indicate success or positive actions.
- **Red**: #DC3545 - Used for errors or negative actions.
- **Yellow**: #FFC107 - Used for warnings or urgent notifications.

## Typography
- **Primary Font**: Open Sans, sans-serif
  - Regular: 16px
  - Medium: 18px
  - Bold: 20px
- **Secondary Font**:Roboto Slab, serif
  - Medium: 24px
  - Bold: 32px

## Component Styles
### Buttons
- **Primary Button**:
  - Background Color: #007BFF
  - Text Color: #FFFFFF
  - Border: None
  - Hover State:
    - Background Color: #6200EA
  - Active State:
    - Background Color: #343A40

- **Secondary Button**:
  - Background Color: #FFFFFF
  - Text Color: #007BFF
  - Border: 1px solid #007BFF
  - Hover State:
    - Background Color: #E9F2FD
  - Active State:
    - Background Color: #D6E4FF

- **Danger Button**:
  - Background Color: #DC3545
  - Text Color: #FFFFFF
  - Border: None
  - Hover State:
    - Background Color: #C82333
  - Active State:
    - Background Color: #A61D29

### Input Fields
- **Text Field**:
  - Background Color: #FFFFFF
  - Text Color: #007BFF
  - Border: 1px solid #CCCCCC
  - Hover State:
    - Border: 1px solid #007BFF
  - Focus State:
    - Border: 1px solid #6200EA

- **Text Area**:
  - Same as text field but with a larger height.

### Cards
- **Base Card**:
  - Background Color: #FFFFFF
  - Box Shadow: 0 4px 8px rgba(0,0,0,0.1)
  - Padding: 1rem

### Text
- **Heading 1**:
  - Font: Open Sans, serif
  - Size: 24px
  - Weight: Bold

- **Heading 2**:
  - Font: Open Sans, serif
  - Size: 20px
  - Weight: Medium

- **Body Text**:
  - Font: Open Sans, serif
  - Size: 16px
  - Weight: Regular

## Layout and Spacing
- **Container Width**: Max-width of 1200px for large screens. Centered content.
- **Spacing**: Use consistent spacing units (e.g., 8px, 16px) to maintain a clean layout.

---

## UX Guidelines
### Navigation
- The navigation bar should be fixed at the top of the screen.
- It should contain links to all major sections and user profiles.

### Forms
- All forms should have clear labels and validation messages.
- Error messages should be displayed prominently and clearly.

### Feedback
- Provide visual feedback for user actions (e.g., button clicks).
- Use notifications or alerts for important information.

### Mobile Responsiveness
- Ensure the app is fully functional on mobile devices.
- Optimize layouts and text sizes for smaller screens.

---

## Conclusion
This style guide helps maintain a cohesive and user-friendly interface across all parts of our task management application. By following these guidelines, we can ensure that users have an enjoyable and efficient experience interacting with our app.
