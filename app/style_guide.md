# Style Guide

## Introduction

This style guide is designed to ensure a consistent and professional user interface for the Call for Tenders API application. It outlines the color scheme, typography, and component styles that should be used throughout the project.

## Color Scheme

1. **Primary Blue**: `#007BFF` - Used for primary actions, buttons, and links.
2. **Secondary Gray**: `#6c757d` - Used for background colors, borders, and text on lighter backgrounds.
3. **Success Green**: `#28a745` - Used to indicate successful operations or positive feedback.
4. **Error Red**: `#dc3545` - Used to indicate errors or negative feedback.

## Typography

1. **Font Family**: Open Sans
   - Regular: `OpenSans-Regular`
   - Bold: `OpenSans-Bold`

2. **Font Sizes**:
   - Header 1 (H1): `2rem`
   - Header 2 (H2): `1.875rem`
   - Header 3 (H3): `1.5rem`
   - Body Text (Body 1): `1rem`
   - Body Text (Body 2): `0.875rem`
