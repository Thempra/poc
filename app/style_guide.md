# Style Guide for the Task Management App

## Overview
This document outlines the style guidelines for the Task Management App. The goal is to ensure a consistent and user-friendly design across all parts of the application, enhancing usability and visual appeal.

## Color Scheme
The color scheme for the app will be minimalistic yet inviting. It will consist of three primary colors:

1. **Primary Blue**: `#007BFF` - This color will be used for buttons, links, and active states to create a sense of action and navigation.
2. **Secondary Grey**: `#6C757D` - This color will be used for background elements, borders, and lesser emphasis text.
3. **Success Green**: `#28A745` - This color will be used to indicate successful actions or completion status.

## Typography
The typography will be consistent throughout the app. The following fonts will be used:

1. **Primary Font**: Open Sans, sans-serif - Used for body text and headings.
   - Regular: 16px
   - Medium: 20px
   - Large: 24px

## Component Styles
### Buttons
- **Primary Button**: 
  - Background Color: `#007BFF`
  - Text Color: `#FFFFFF`
  - Hover State: Background Color: `#0056b3`
  - Active State: Background Color: `#004085`

- **Secondary Button**:
  - Background Color: `#6C757D`
  - Text Color: `#FFFFFF`
  - Hover State: Background Color: `#5a6268`
  - Active State: Background Color: `#495057`

### Inputs
- **Input Field**:
  - Border Color: `#ced4da`
  - Focus Border Color: `#007BFF`
  - Placeholder Color: `#6c757d`
  - Text Color: `#212529`

- **Text Area**:
  - Border Color: `#ced4da`
  - Focus Border Color: `#007BFF`
  - Placeholder Color: `#6c757d`
  - Text Color: `#212529`

### Headers
- **H1**: Regular Open Sans, size 28px
- **H2**: Medium Open Sans, size 24px
- **H3**: Medium Open Sans, size 20px

### Links
- **Default Link**:
  - Text Color: `#007BFF`
  - Hover State: Text Color: `#0056b3`

## Layout and Spacing
- **Container Width**: Max width of 1200px, centered on the page.
- **Grid System**: Use a 12-column grid system for layout consistency.
- **Spacing**:
  - Margin: `.25rem` (4px), `.5rem` (8px), `1rem` (16px), `1.5rem` (24px)
  - Padding: `.25rem` (4px), `.5rem` (8px), `1rem` (16px), `1.5rem` (24px)

## Responsive Design
The app should be designed to be responsive and look good on various devices, including desktops, tablets, and smartphones.

By following these style guidelines, we ensure a cohesive and user-friendly design for the Task Management App.
